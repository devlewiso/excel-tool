Excel File Visualizer

Descripción

Este proyecto es una aplicación web que permite a los usuarios cargar archivos de Excel (.xlsx, .xls), visualizar los datos en una tabla y generar gráficos a partir de ellos. Se utiliza SheetJS (xlsx) para procesar los archivos de Excel y Chart.js para la representación gráfica de los datos.

Características

Carga de archivos Excel (.xlsx, .xls).

Visualización de datos en una tabla interactiva.

Funcionalidad de búsqueda dentro de los datos cargados.

Representación de datos en gráficos dinámicos usando Chart.js.

Opciones para limpiar la búsqueda y eliminar los datos cargados.

Tecnologías Utilizadas

HTML5: Estructura de la aplicación.

CSS3: Estilos y diseño responsivo.

JavaScript (ES6): Manipulación del DOM y gestión de eventos.

SheetJS (xlsx): Procesamiento de archivos Excel.

Chart.js: Generación de gráficos dinámicos.

Uso

Abre el archivo index.html en un navegador web compatible.

Carga un archivo de Excel utilizando el botón Choose an Excel file.

Observa los datos en la tabla y el gráfico generado automáticamente.

Utiliza la barra de búsqueda para filtrar datos en la tabla.

Usa los botones Clear Search y Clear Data para limpiar la búsqueda o eliminar los datos cargados.

Dependencias

Este proyecto utiliza bibliotecas externas que se cargan desde un CDN:

SheetJS (xlsx)

Chart.js

Personalización

Si deseas modificar la apariencia de la aplicación, puedes editar los estilos en la sección <style> dentro del archivo index.html. Para cambiar el comportamiento de la carga y visualización de datos, edita el código JavaScript en la sección <script>.
